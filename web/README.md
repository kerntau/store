# Web

React + Vite + TypeScript storefront and admin console.

## Commands

```powershell
npm install
npm run dev
npm run lint
npm run build
```

## Structure

- `src/public/` buyer storefront, product detail, order query, order result
- `src/admin/` admin shell, dashboard, products, cards, orders
- `src/api/` API client and shared response types
- `public/` static runtime assets such as favicon and background image
